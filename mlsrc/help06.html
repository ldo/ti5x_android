ML-06 — Complex Trigonometric Functions

First, enter the complex number X which is the operand for the remaining operations: enter the real part of X, then press  A , then enter the imaginary part, and press  A  again.

(This is the same initial entry procedure as for ML-04 and ML-05, and indeed the first or only operand entered for any of these programs may be used for any of the others.)

The remaining functions return the real part of the result in the display, and the imaginary part in the T register, and also replace the X operand with the result, ready for further operations:

 B 
Compute sin X.
2nd B´
Compute arcsin X.
 C 
Compute cos X.
2nd C´
Compute arccos X.
 D 
Compute tan X.
2nd D´
Compute arctan X.

Note that all computations are done in the currently-selected angle units: degrees, radians or gradians.

Formulas Used

sin X = sin a cosh b + i cos a sinh b

cos X = cos a cosh b - i sin a sinh b

tan X = sin X ÷ cos X

arcsin X = arcsin B + i ln [A + (A2 - 1)0.5]

arccos X = arccos B - i ln [A + (A2 - 1)0.5]

arctan X = 0.5 arctan [2a ÷ (1 - a2 - b2)] + 0.25i ln [(a2 + (b + 1)2) ÷ (a2 + (b - 1)2)]

where

X = a + bi (a and b both real)

A = (((a + 1)2 + b2)0.5 + ((a - 1)2 + b2)0.5) ÷ 2

B = (((a + 1)2 + b2)0.5 - ((a - 1)2 + b2)0.5) ÷ 2

Registers used: 01-07.
