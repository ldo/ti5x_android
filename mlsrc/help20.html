This program calculates the number of days between any two Gregorian calendar dates after 1582. It can also calculate the day of the week corresponding to a date.

To compute the number of days between two dates:

  • Enter the first date: yyyy.mmdd  A 
  • Enter the second date: yyyy.mmdd  B 
  • Display the number of days in-between:  C 

You can re-enter either of the first two values while keeping the existing value of the other one, then press  C  to compute the new number of days.

To compute the day of week corresponding to a date:

  • Enter the date: yyyy.mmdd then press  D  to display a number in the range [0 .. 6], with 0 representing Sunday .. 6 representing Saturday.

Registers used: 01-05.
