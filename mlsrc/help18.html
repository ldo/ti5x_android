ML-18 — Compound Interest

Enter 3 of the following 4 quantities, in any order:

  • N  A  — enter number of periods
  • %I  B  — enter percent interest rate per period
  • PV  C  — enter present value
  • FV  D  — enter future value

To compute the missing value, enter 0 before pressing the corresponding key, i.e.

  • 0  A  — compute number of periods
  • 0  B  — compute percent interest rate per period
  • 0  C  — compute present value
  • 0  D  — compute future value

The following functions assume that N and %I have already been computed or entered (i = %I ÷ 100):

  • 2nd A´ — Sinking Fund, Sn|i = [(1 + i)N - 1] ÷ i
  • 2nd B´ — Annuity Due/FV, (1 + i)Sn|i = [(1 + i)N + 1 - (1 + i)] ÷ i
  • 2nd C´ — Ordinary Annuity/PV, an|i = [1 - (1 + i)-N] ÷ i
  • 2nd D´ — Annuity Due/PV, (1 + i)an|i = [(1 + i) - (1 + i)1 - N] ÷ i

Registers used: 01-04, 08, 09.
